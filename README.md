# ☕ Java - Programação Orientada a Objetos

## 📖 Sobre o Repositório

Este repositório reúne projetos desenvolvidos em Java com foco no estudo de **Programação Orientada a Objetos (POO)**.

O objetivo foi aplicar os principais conceitos da linguagem por meio de projetos práticos, evoluindo gradualmente da implementação de classes e objetos para sistemas mais estruturados.

Cada projeto possui seu próprio README, contendo uma descrição detalhada, funcionalidades e conceitos utilizados durante o desenvolvimento.

---

# 🎯 Objetivos

- Praticar Programação Orientada a Objetos.
- Desenvolver projetos utilizando Java.
- Aplicar conceitos de encapsulamento e abstração.
- Melhorar a organização e estrutura do código.
- Consolidar conhecimentos através de projetos práticos.

# 📈 Evolução

Este repositório representa minha evolução no aprendizado de Java.

Os primeiros projetos tiveram como foco a compreensão dos conceitos fundamentais da Programação Orientada a Objetos. Com o avanço dos estudos, passei a desenvolver aplicações mais estruturadas, incorporando boas práticas de organização do código e, posteriormente, integração com banco de dados utilizando JDBC em repositórios específicos.
