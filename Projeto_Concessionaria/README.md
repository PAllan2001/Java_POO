# 🚗 Sistema de Concessionária (Java)

Projeto em Java simulando o gerenciamento de veículos em uma concessionária, com foco em organização de código e manipulação de dados.

---

## 🎯 Objetivo

Criar um sistema simples para cadastro e gerenciamento de veículos, aplicando conceitos de POO e estrutura de dados.

---

## ⚙️ Funcionalidades

- Cadastro de veículos (Carro e Moto)
- Listagem de veículos
- Busca por placa
- Remoção de veículo
- Validação de placa única (não permite duplicatas)

---

## 🧱 Estrutura do Projeto
src/
├── model/
│ ├── Veiculo.java
│ ├── Carro.java
│ └── Moto.java
├── service/
│ └── VeiculoService.java
└── app/
└── Main.java

---

## 🧠 Conceitos Aplicados

- Programação Orientada a Objetos (POO)
- Herança (Veículo → Carro/Moto)
- Polimorfismo
- Encapsulamento
- Separação de responsabilidades (Model / Service / Main)
- Uso de ArrayList para gerenciamento de dados
- Validação de regras de negócio (placa única)

---

📚 Aprendizados

- Organização de código em camadas
- Implementação de operações básicas (CRUD)
- Aplicação de validações de negócio
- Simulação de um sistema real de gerenciamento
